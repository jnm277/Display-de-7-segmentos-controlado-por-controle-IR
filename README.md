# Display-de-7-segmentos-controlado-por-controle-IR

Display de 7 segmentos controlado por controle remoto IR, feito usando a biblioteca do DIYables.

# COMPONENTES:
> ARDUINO UNO R3
> Display 7 segmentos
> Sensor IR 38kHz
> controle remoto 17 teclas

<br>

# CONEXÕES DO DISPLAY:
>GPIO 2 - SEG A
>GPIO 3 - SEG B
>GPIO 4 - SEG C
>GPIO 5 - SEG D
>GPIO 6 - SEG E
>GPIO 7 - SEG F
>GPIO 8 - SEG G

<br>

# CONEXÕES DO SENSOR IR 38kHz
>GPIO 9 - Pino de saída do sensor
>5V - Vcc do sensor
>GND - GND do sensor
